# WezTerm Lua設定 - キーバインド一覧（Mac版）

**Leaderキー**: `Ctrl + q` (タイムアウト: 2秒)

---

## ワークスペース操作

| キー | 説明 |
|------|------|
| `Leader + w` | ワークスペースの切り替え |
| `Leader + $` | ワークスペース名の変更 |
| `Leader + Shift + w` | 新しいワークスペースの作成 |

---

## タブ操作

| キー | 説明 |
|------|------|
| `Cmd + t` | 新しいタブを作成 |
| `Cmd + w` | 現在のタブを閉じる（確認あり） |
| `Ctrl + Tab` | 次のタブに移動 |
| `Ctrl + Shift + Tab` | 前のタブに移動 |
| `Leader + {` | タブを左に移動 |
| `Leader + }` | タブを右に移動 |
| `Cmd + 1` ~ `Cmd + 8` | 指定した番号のタブに切り替え |
| `Cmd + 9` | 最後のタブに切り替え |

---

## ペイン操作

| キー | 説明 |
|------|------|
| `Leader + d` | ペインを縦に分割 |
| `Leader + r` | ペインを横に分割 |
| `Leader + x` | 現在のペインを閉じる（確認あり） |
| `Leader + h` | 左のペインに移動 |
| `Leader + l` | 右のペインに移動 |
| `Leader + k` | 上のペインに移動 |
| `Leader + j` | 下のペインに移動 |
| `Ctrl + Shift + [` | ペイン選択モード |
| `Leader + z` | ペインのズーム切り替え（全画面表示） |

---

## ペインサイズ調整（キーテーブルモード）

`Leader + s` でサイズ調整モードに入る

| キー | 説明 |
|------|------|
| `h` | 左に拡大 |
| `l` | 右に拡大 |
| `k` | 上に拡大 |
| `j` | 下に拡大 |
| `Enter` | サイズ調整モードを終了 |

---

## ペイン移動（キーテーブルモード）

`Leader + a` で移動モードに入る（1秒タイムアウト）

| キー | 説明 |
|------|------|
| `h` | 左のペインに移動 |
| `l` | 右のペインに移動 |
| `k` | 上のペインに移動 |
| `j` | 下のペインに移動 |

---

## コピーモード（Vim風操作）

| キー | 説明 |
|------|------|
| `Leader + [` | コピーモードを起動 |

### コピーモード内の操作

#### 移動

| キー | 説明 |
|------|------|
| `h` / `j` / `k` / `l` | カーソル移動（左/下/上/右） |
| `w` | 次の単語に移動 |
| `b` | 前の単語に移動 |
| `e` | 単語の末尾に移動 |
| `0` | 行の先頭に移動 |
| `^` | 行の最初の文字に移動 |
| `$` | 行末に移動 |
| `g` | スクロールバックの先頭に移動 |
| `G` | スクロールバックの最後に移動 |
| `H` | ビューポート上部に移動 |
| `M` | ビューポート中央に移動 |
| `L` | ビューポート下部に移動 |

#### ジャンプ

| キー | 説明 |
|------|------|
| `f` + 文字 | 次の指定文字にジャンプ |
| `F` + 文字 | 前の指定文字にジャンプ |
| `t` + 文字 | 次の指定文字の直前にジャンプ |
| `T` + 文字 | 前の指定文字の直後にジャンプ |
| `;` | ジャンプを繰り返す |

#### スクロール

| キー | 説明 |
|------|------|
| `Ctrl + f` | 1ページ下にスクロール |
| `Ctrl + b` | 1ページ上にスクロール |
| `Ctrl + d` | 半ページ下にスクロール |
| `Ctrl + u` | 半ページ上にスクロール |

#### 選択

| キー | 説明 |
|------|------|
| `v` | セル選択モード（文字単位） |
| `V` | 行選択モード |
| `Ctrl + v` | ブロック選択モード（矩形選択） |
| `o` | 選択範囲の反対側に移動 |
| `O` | 選択範囲の反対側（水平）に移動 |

#### コピー・終了

| キー | 説明 |
|------|------|
| `y` | クリップボードにコピー |
| `Enter` | コピーしてモード終了 |
| `q` / `Esc` / `Ctrl + c` | コピーモード終了 |

---

## クリップボード

| キー | 説明 |
|------|------|
| `Cmd + c` | クリップボードにコピー |
| `Cmd + v` | クリップボードから貼り付け |

---

## フォント

| キー | 説明 |
|------|------|
| `Ctrl + +` | フォントサイズを大きくする |
| `Ctrl + -` | フォントサイズを小さくする |
| `Ctrl + 0` | フォントサイズをリセット |

---

## その他

| キー | 説明 |
|------|------|
| `Cmd + p` | コマンドパレットを表示 |
| `Ctrl + Shift + p` | コマンドパレットを表示 |
| `Opt + Enter` | フルスクリーン切り替え |
| `Ctrl + Shift + r` | 設定を再読み込み |
| `Shift + Enter` | 改行を送信 |

---

## キー表記について

- `Leader` : `Ctrl + q` （Leaderキー）
- `Cmd` : Command キー
- `Ctrl` : Control キー
- `Opt` : Option キー
- `Shift` : Shift キー
- `Enter` : Enter/Return キー
- `Esc` : Escape キー
- `Tab` : Tab キー

---

## その他の設定

- **デフォルトのキーバインド**: 無効化されています
- **キーテーブル表示**: アクティブな場合、右側のステータスエリアに表示
- **タブバー**: タブが1つの場合は自動的に非表示
- **背景透過度**: 85%
- **背景ブラー**: 20（macOS）
- **フォントサイズ**: 15.0
